## FindCourse

A PHP-based web platform that helps students discover and explore courses offered by private instructors. Built with PHP, MySQL, and vanilla CSS.

---

## Features

**For Visitors**
- Browse all available courses
- Search and filter by name, mode (Online/Offline), and location
- View full course details including instructor info, subjects, dates, and price

**For Users**
- Create an account and log in securely
- Rate courses with a 1–5 star system
- View and update profile (email, password)
- Send messages to the admin via the contact form

**For Admins**
- Secure admin login
- Dashboard with stats (total courses, users, messages)
- Add, edit, and delete courses (with image upload)
- View all user messages
- Add new admin accounts

---
## Tech Stack

- **Frontend:** HTML, CSS (custom), Google Fonts
- **Backend:** PHP 8.x
- **Database:** MySQL (via MySQLi)
- **Local Server:** MAMP

---

## Installation

### Requirements
- MAMP (or XAMPP / any local PHP server)
- PHP 8.0+
- MySQL
